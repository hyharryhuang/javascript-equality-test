javascript-equality-test
========================

Testing various different javascript equalities.

Results:

    Javascript Equality Test

    "1" == 1 is: true  
    "1" === 1 is: false
    NaN == NaN is: false
    NaN === NaN is: false
    undefined == undefined is: true
    null == undefined is: true
    null === undefined is: true
    False == 0 is: true
    True == 1 is: true
    True == 2 is: false
    null == 0 is: false
    undefined == 0 is: false
